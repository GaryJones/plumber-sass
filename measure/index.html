This tool helps you measure the baseline position of a font to use with Plumber

1. Look up the exact font you’d like to use on Google Fonts and paste the embed code here:

2. Use the up and down keys on your keyboard to move the dashed line until it just disappears under the base of the letter. Hold down the Shift key to step 10 pixels.

3. Done! Use the number on the line for this font with Plumber. Don’t hesitate to create a pull request on the baseline list with your measurement.
